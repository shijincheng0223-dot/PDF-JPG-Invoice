# PDF Go Pro - 高清发票转换器

PDF Go Pro 是一款专为发票处理设计的本地化、高清 PDF 转图片工具。它可以自动解析发票中的金额和发票号码，并以此智能命名导出的图片。

## ✨ 主要功能

- **高清转换**：支持 1.5x、2x、3x 等多种渲染质量，确保发票文字清晰可辨。
- **智能重命名**：自动从 PDF 内容中提取“合计金额”和“发票号码”，转化为 `发票 [金额]+[号码].jpg` 格式。
- **防止文件名冲突**：批量导出时，若检测到重名文件，将自动添加 (1), (2) 等后缀。
- **隐私安全**：所有处理过程均在浏览器本地完成，发票数据不会上传到任何服务器。
- **批量导出**：支持一次性处理多个 PDF 文件，并一键打包导出 ZIP。

## 🚀 如何使用

1. 在浏览器中打开 `index.html`。
2. 选择所需的“渲染质量”（默认为高清 2x）。
3. 将一个或多个发票 PDF 文件拖入虚线框内，或点击框体选择文件。
4. 等待转换完成后，预览生成的图片及解析出的信息。
5. 点击“导出 ZIP 压缩包”批量下载，或点击单张图片进行预览和下载。

## 🛠️ 技术栈

- **Tailwind CSS**：现代化的 UI 设计。
- **PDF.js**：强大的 PDF 渲染与文本解析引擎。
- **JSZip**：实现本地文件打包压缩。

PDF to JPG Go - 发票智能命名版：项目需求文档

1. 项目背景

在日常财务报销或文件管理中，用户经常需要将 PDF 格式的电子发票转换为图片格式。手动一张张截图并按照金额命名耗时耗力。本项目旨在提供一个极简、安全、自动化的工具，实现一键批量转换并智能命名。

2. 核心功能需求

2.1 高清转换 (Core Conversion)

批量处理：支持一次性拖入多个 PDF 文件。

多页支持：自动解析多页 PDF，并将每一页转换为独立的图片。

高清渲染：支持渲染倍率选择（1.5x、2x、3x），确保发票上的印章和文字清晰可见。

格式支持：导出为 JPEG 或 PNG 格式。

2.2 智能命名 (Intelligent Naming)

自动 OCR 扫描：利用 PDF.js 提取文档文字层信息。

发票特征识别：通过正则表达式自动提取发票中的：

金额（匹配“合计”、“金额”、“¥”等关键字后的数值）。

发票编号（匹配 8 位或 20 位发票号码）。

命名规范：转换后的图片自动重命名为：发票 [金额]+[编号].jpg。

容错处理：若识别失败，则回退使用原文件名，确保流程不中断。

2.3 用户体验 (User Interface)

视觉风格：现代 SaaS 风格设计，采用毛玻璃效果、渐变背景及 Inter 字体。

交互反馈：

带有外发光效果的实时进度条。

转换成功后的图片预览网格（Gallery）。

悬浮显示识别出的文件名。

一键打包：集成 JSZip，支持将所有转换后的图片一键打包成 ZIP 文件下载。

2.4 安全与隐私 (Privacy & Security)

本地化处理：所有文件解析和图片渲染均在用户浏览器本地完成。

零服务器上传：严禁将发票数据上传至任何后端服务器，确保隐私安全。

3. 技术栈

前端框架：Tailwind CSS (UI 布局与样式)。

PDF 引擎：PDF.js (Mozilla 开源，用于解析与渲染)。

压缩工具：JSZip (用于客户端生成压缩包)。

部署方式：单文件 HTML，支持 GitHub Pages 托管。

4. 后续优化方向 (Roadmap)

纯图片识别：针对扫描件（无文字层），引入 Tesseract.js 进行本地 OCR 识别。

多模板支持：增加针对滴滴行程单、火车票、京东小票等不同格式的特殊识别逻辑。

云端备份：在用户授权的前提下，提供保存到私人云存储的功能。

深色模式：适配系统的 Dark Mode，提升夜间使用体验。
